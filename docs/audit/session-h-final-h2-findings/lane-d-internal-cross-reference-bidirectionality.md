- Research directory (`docs/research/`) was treated as a trusted source, not an audit target (§4 / §9.3).

- Primitive tightening: "carries the claim" = (a) verbatim substring OR (b) semantic-equivalent phrasing with cited alignment. Escalation when in doubt: DRIFT + let H3 triage decide.
- Every finding carries verbatim `claim-text` + verbatim `evidence-quote` per §8.1.
